# Design Brief / Brand & Style Guide

## Brand direction

StreamYolo is a chromeless, room-first live platform. The product should feel like glass over a room, not a frame around it.

## Core design principle

```txt
The room is the brand.
The stream is the interface.
The platform disappears.
```

## Visual personality

- Neutral.
- Chromeless.
- Dark but not stylized.
- Video/thumbnail first.
- Sparse, white-on-dark floating controls.
- Brand appears only in masthead, primary buttons, token controls, and checkout moments.

## Avoid

- Neon casino styling.
- Overdesigned SaaS dashboard look.
- Porn-site red/black cliché.
- Constant gradients.
- Heavy sidebars inside rooms.
- Decorative panels that compete with the room.

## Surfaces

### Room surface

Primary experience. Video, thumbnail, screenshot, or fallback image fills the visual space.

### Overlay controls

Small floating controls on dark translucent backgrounds. Fade in/out based on user activity.

### Utility drawers

Chat, wallet, menu, private requests, creator settings. Drawers appear when needed and do not dominate the room by default.

## Brand voice

- Direct.
- Clear.
- Slightly playful in room prompts.
- Never corporate.
- Never vulgar by default.
- Creator-configurable labels carry most personality.
