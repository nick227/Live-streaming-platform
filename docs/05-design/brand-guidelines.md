# Brand Guidelines

## Working brand

StreamYolo

## Brand role

The brand should be present but quiet. It should not compete with the creator's room.

## Logo usage

MVP can use a wordmark only.

Rules:
- Small masthead wordmark.
- Do not place large logos over room video.
- Do not watermark streams in MVP.
- Use neutral white wordmark on dark background.

## Voice and tone

System copy:
- Clear.
- Direct.
- Calm.
- Slightly playful in auto-messages.

Examples:

```txt
Complete room setup to go live.
Make this frame my thumbnail.
Private is available: 150 tokens/min.
Your balance supports about 8 minutes.
Menu is open — choose a request.
```

Avoid:
- Corporate boilerplate.
- Overly explicit default copy.
- Fake urgency.
- Casino-like language.

## Imagery style

- Creator-chosen thumbnails and covers carry the room identity.
- The platform should not impose a strong visual theme on creator imagery.
- Use neutral placeholders when imagery is missing.

## UI branding

Brand may appear in:
- Masthead.
- Primary buttons.
- Token purchase page.
- Empty states.
- Loading screens.

Brand should not dominate:
- Public room video.
- Private session video.
- Creator thumbnails.
- Chat content.
